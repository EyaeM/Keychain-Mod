# A Simple mod to add keychains that hold Supplementaries keys

## Features
- Adds a keychain item to store multiple Supplementaries keys
- Shift + Right-click to open the keychain and manage your keys
- Use keys directly from the keychain without removing them
## Requirements
- Minecraft 1.21.1 (feel free to port this to other version as this is open source)
- Supplementaries (required)
- Curios (Optional)
